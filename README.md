# picoTracker-playground

This is a collection of community contributed alternative firmwares for the RP2040 "v2 PCB" picoTracker device.

Please note: the contents of this repository are intended for educational and recreational purposes. 

The firmwares are contributed by members of the picoTrcker community and are ***not*** supported by xiphonics and are installed and used at the device owners own risk.

See the Readme in each subdirectory for instructions on how to build, install and use each different firmware.


## Firmwares

### pT_dx7

A 6 voice emulator of the classic DX7 synth with a reverb master output effect.
